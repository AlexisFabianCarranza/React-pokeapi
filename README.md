## Step by step

- Environtment setup: <p> 
You need to install NodeJS and NPM.  https://nodejs.org/es/download/
- Clone the repository: <p> 
`git clone https://github.com/AlexisFabianCarranza/React-pokeapi.git`
- Install dependencies: <p> 
`yarn`
- Run project: 
`yarn start`
